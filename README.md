# 插件开发模板含内嵌CSS

#### 项目介绍
插件开发模板含内嵌CSS，若不包含CSS或者将CSS写入脚本之中，建议使用rollup


#### 安装教程

    1、将项目拷贝至本地；
    2、安装依赖包:
        npm: npm install
        cnpm: cnpm i
        yarn: yarn install

#### 使用说明

    打包目前项目:
        npm run build

    进入开发环境:
        npm run dev

    清除当前已打包项目
        npm run clean

    查看任务树
        npm run tree
